## Жёсткие запреты (мгновенный результат)

| Ситуация | Результат |
|---|---|
| Собака проявляет агрессию, атакует собаку группы или человека при докладе при заходе | **Дисквалификация** |
| Обозначение: собака вообще не указывает на статиста | **0 баллов за обозначение** |
| Обозначение: собака сразу уходит от статиста после обозначения | **0 баллов за обозначение** |
| Любая помощь собаке со стороны проводника/статиста/прокладчика трейла во время обозначения | **0 баллов за обозначение** |
| Собака не выходит из тоннеля, войдя в него | **Недостаточно** |
| Собака не остаётся после AC/VC на выходе из тоннеля | **Недостаточно** |
| Собака спрыгивает с мостика/лестницы/качелей в первой половине | **0 баллов** |
| Собака спрыгивает с мостика/лестницы/качелей после первой половины | **Недостаточно** |
| Собака спрыгивает со стола в упражнении «Смена положений» | **Недостаточно** |
| Собака проваливается между перекладинами и сама восстанавливается | **Недостаточно** |
| Собака проваливается между перекладинами и не может восстановиться | Считается как спрыгивание в этой точке |
| Статист не найден при поиске в природной среде или на техногенном завале | **Не может пройти** (даже с максимальной частичной оценкой) |
| Средняя точка не показана в «Управлении направлением на дистанции» | **Недостаточно** |
| Неверная последовательность столов в «Управлении направлением на дистанции» | **Недостаточно** |
| Собака спрыгивает в упражнении «Перенос и передача» | **Недостаточно** |

## Фиксированные снятия баллов

| Ситуация | Снятие |
|---|---|
| Неверная позиция после команды «Сидеть» в упр. 3 (лежать/стоять вместо сидеть) | −5 б |
| Собака стоит или сидит в зоне укладки (не полная укладка), но остаётся | −5 б |
| Смена положений: неверная/пропущенная позиция или собака не удерживается | −4 б (F/FL/T/L/MT) / −3 б (W) |
| Дистанция превышает 10 шагов при подзыве в «Управлении на дистанции» | максимум «Удовлетворительно» за эту часть |
| Собака покидает помощника и возвращается до подзыва в «Переносе» (уровень B) | максимум «Удовлетворительно» |
| 1-е ложное обозначение в природной среде/на техногенном завале уровня A/B | −40 б |
| 1-е ложное обозначение в предварительных тестах V | −20 б |
| Лестница: собака спрыгивает после последней перекладины | максимум «Удовлетворительно» |
| Обозначение: собака один раз покидает место обозначения и самостоятельно возвращается | максимум «Удовлетворительно» |
