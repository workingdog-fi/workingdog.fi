## Роли и участники

| Сокр. | Расшифровка |
|---|---|
| D | Dog — собака |
| DH | Dog Handler — проводник |
| TJ | Trial Judge — судья |
| TC | Trial Chairman — председатель испытаний |
| HP | Hilfsperson — помощник в упражнении «Перенос и передача» |
| VP | Versuchsperson — статист (человек, которого ищут) |
| FHL | Fährtenleger — прокладчик следа (дисциплина F) |
| TL | Trail Layer — прокладчик трейла (дисциплина MT); также Team Leader — руководитель команды в общем контексте SAR |
| RHT | RettungsHundTeam — команда поисково-спасательной собаки (проводник + собака) |
| RDT | Rescue Dog Team — то же, что RHT; обе формы встречаются в документах |

## Дисциплины (RH = Rettungshund)

| Сокр. | Расшифровка |
|---|---|
| RH-F | Fährtenhund — следовая собака |
| RH-FL | Flächensuchhund — собака для поиска в природной среде |
| RH-T | Trümmersuchhund — собака для поиска на техногенном завале |
| RH-L | Lawinensuchhund — лавинная собака |
| RH-MT | Mantrailing — мантрейлинг |
| RH-W | Wasserrettungshund — водноспасательная собака |

## Примечания

- **VS** встречается один раз в файле упражнений (Упр. 4 «Управление на дистанции»), где по смыслу подразумевается **VC** — вероятно, опечатка в исходном документе.
- **TL** имеет двойное значение: Trail Layer — прокладчик трейла в дисциплине MT; Team Leader — руководитель команды в общем организационном контексте. В таблице правил обозначения («Любая помощь со стороны DH/VP/TL → 0 баллов») TL означает человека, которого ищут в дисциплине MT.
- **RHT / RDT**: оба обозначают рабочую команду (проводник + собака). RHT — немецкая форма, RDT — английская. В документах используются как синонимы.
